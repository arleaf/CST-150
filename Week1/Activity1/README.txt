# Activity 1 ReadMe
This program displays my name and location.

# Controls Used (3 controls minimum as per rubric)
The controls used in this program include
  -Labels
  -Buttons
  -Textbox
 
# Onchange
  In the program there is a label (info) which starts off with no text. Once a button is clicked, the label should display its' corresponding text. 
  If location is clicked, the label should display " I live in Phoenix, Arizona"
  If name is clicked, the label should display " My name is Arleth Martinez".
