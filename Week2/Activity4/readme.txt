# Activity 4 ReadMe
This program will convert seconds into days,hours, minutes accordingly.
It will aslo disply remaining seconds after conversion of the above.

# Controls Used
The controls used in this program include
  -Labels
  -Buttons
  -Textbox
 
# Onchange
  Once the button calculate equivelent is clicked, it will convert those seconds into days if greater than 86400, hours if greater than 3600, minutes if greater than 60 
  or just display the seconds else. TryParse was used to to parse textbox input and inputvalidation was used to ensure postive integer was entered.
